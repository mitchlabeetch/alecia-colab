# Changelog

All notable changes to Alecia Colab will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Complete rebranding from Novel template to Alecia Colab
- M&A-specific slash commands for quick document creation
- Deal summary template with strategic rationale sections
- Due diligence checklist template with financial, legal, and operational categories
- Valuation analysis template with multiple methodologies
- Integration plan template with Day 1 priorities and synergy tracking
- Company profile template for target analysis
- Meeting notes template with action items
- Deal pipeline visualization with stage tracking
- Document categorization system (deals, companies, research, etc.)
- Enhanced menu with M&A-specific quick actions
- Export functionality for documents (Markdown format)
- Custom color scheme aligned with professional M&A branding
- Improved save status indicator with visual feedback
- Template library system for easy reuse
- Comprehensive README with Alecia-specific documentation
- Contributing guidelines for community participation
- Enhanced security documentation

### Changed
- Updated all branding from "Novel" to "Alecia Colab"
- Redesigned navigation header with Alecia branding
- Replaced default editor content with M&A knowledge base template
- Enhanced save status with color-coded visual feedback
- Updated metadata and descriptions for M&A focus
- Improved error handling in editor component

### Removed
- External template references to steven-tey/novel
- Generic "Deploy to Vercel" and feedback mechanisms
- French language elements (replaced with English)
- Twitter/X integration from slash commands
- Generic demo content and examples

### Technical
- Added @radix-ui/react-tabs for tabbed interface
- Created Badge, Card, and Tabs UI components
- Implemented document sidebar component
- Added deal pipeline component with stage filtering
- Created templates library with categorization
- Enhanced TypeScript types for templates and deals

## [0.1.0] - 2026-01-15

### Initial Release
- Fork from Novel open-source editor
- Integration of Clerk authentication with Microsoft OAuth
- Basic WYSIWYG editing capabilities
- AI-powered content suggestions
- Image upload functionality
- Code syntax highlighting
- Mathematical expression support (LaTeX)
- Task lists and checklists
- Dark mode support
