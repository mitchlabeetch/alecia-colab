# Alecia Design System - UI Patterns

## Intake/Triage (from Plane)
- **Inbox**: Section for new leads.

## Settings Modal (from Plane)
- **Profile Editing**: In a modal, not a page.

## Block Editor (from AFFiNE)
- **Études de Marché**: Use for Market Studies.

## Dense Lists (from Odoo)
- **Rows**: py-2
- **Status Pills**: Colored.
- **Menu**: "Three-dot" menu on right.

## Command Palette (from OpenBB)
- **Navigation**: Cmd+K is the primary navigation tool.

## Data Grids (from OpenBB)
- **Numbers**: Monospace fonts.
- **Currency**: Right-aligned.

## Audit Trail (from Pipedream)
- **Historique d'activité**: Collapsible on detail pages.
