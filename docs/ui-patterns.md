# Alecia Design System - Deep Inspiration Codex

## Plane
*   **Intake/Triage:** "Inbox" section for new leads.
*   **Settings Modal:** Profile editing in a modal, not a page.

## AFFiNE
*   **Block Editor:** Use for "Études de Marché" (Market Studies).

## Odoo
*   **Dense Lists:** py-2 rows, colored status pills, "three-dot" menu on right.

## OpenBB
*   **Command Palette (Cmd+K):** Primary navigation tool.
*   **Data Grids:** Monospace fonts for numbers, right-aligned currency.

## Pipedream
*   **Audit Trail:** Collapsible "Historique d'activité" on detail pages.
